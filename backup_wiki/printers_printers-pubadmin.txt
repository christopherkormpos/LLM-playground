====== Εγκατάσταση Σε PUBADMIN ======

===== Εγκατάσταση  USB PRINTER Σε TPCMOD PC =====

==== Περίπτωση 1 - Χρήση Drivers Διαθέσιμων Στα Windows ====

Επιχειρούμε να εγκαταστήσουμε τον εκτυπωτή μέσω των drivers που είναι διαθέσιμοι ήδη στα windows.
  - Devices and Printer 
  - Add a Printer 
  - Add a local printer
  - Use an existing port (USB001 - Virtual Printer Port for USB)
  - Διαλέγουμε εκτυπωτή από την λίστα & ακολουθούμε βήματα installer.


==== Περίπτωση 2 - Χρήση Drivers Από Εξωτερικό Σκληρό / USB Flash ====

Επιχειρούμε να εγκαταστήσουμε τον εκτυπωτή μέσω των drivers που είναι διαθέσιμοι στον εξωτερικό σκληρό δίσκο που υπάρχει στο Helpdesk. Βρίσκουμε τους Drivers του εκτυπωτή και τρέχουμε το αντίστοιχο Install. 
**Συνδέουμε το USB μόνο όταν μας ζητηθεί από την εγκατάσταση.**

==== Περίπτωση 3 - Χρήση Drivers Από Διαδίκτυο ====

Ψάχνουμε τους drivers στο ίντερνετ κι αφού κατεβάσουμε τους σωστούς κάνουμε την εγκατάσταση.

===== Εγκατάσταση Network PRINTER Σε TPCMOD PC =====

Εκτυπώνουμε τη σελίδα με τη MAC Address του εκτυπωτή και τη δίνουμε στα Δίκτυα για τη δήλωσή του.

Μας επιστρέφεται η σελίδα με την IP του εκτυπωτή την οποία θα πάρει με DHCP ο εκτυπωτής (192.168.35.xxx).

Αφού έχουμε την IP κάνουμε εγκατάσταση του Printer με έναν από τους παρακάτω τρόπους:

==== Περίπτωση 1 - Χρήση Drivers Διαθέσιμων Στα Windows ====

Επιχειρούμε να εγκαταστήσουμε τον εκτυπωτή μέσω των drivers που είναι διαθέσιμοι ήδη στα windows.
  - Devices and Printer 
  - Add a Printer 
  - Add a network, wireless or Bluetooth printer
  - The printer that I want isn’t listed 
  - Add a printer using TCP/IP address or hostname 
  - Γράφουμε τη IP που δώσαμε στον εκτυπωτή 
  - Αφού το PC βρεί τον εκτυπωτή, τον επιλέγουμε μαζί με κατάλληλους drivers από την λίστα και ακολουθούμε την διαδικασία

==== Περίπτωση 2 - Χρήση Drivers Από Εξωτερικό Σκληρό / USB Flash ====

Βρίσκουμε τους σωστούς drivers, εκτελούμε και ακολουθούμε τα βήματα εισάγοντας την IP που δώσαμε στον εκτυπωτή 

==== Περίπτωση 3 - Χρήση Drivers Από Διαδίκτυο ====

Ψάχνουμε τους drivers στο ίντερνετ κι αφού κατεβάσουμε τους σωστούς κάνουμε την εγκατάσταση.
