====== Εγκατάσταση Πιστοποιητικών e-Gate ======

Η συγκεκριμένη εφαρμογή επιτρέπει στους χρήστες της να επικοινωνούν με τις πύλες του στρατοπέδου και να στέλνουν άδειες εισόδου μέσω mod υπολογιστών. 

Είναι web based εφαρμογή και για να έχουν πρόσβαση οι χρήστες απαιτείται και η εγκατάσταση δύο πιστοποιητικών.

<callout type="info">
Τα πιστοποιητικά βρίσκονται στο δικτυακό φάκελο (\\10.0.0.15\sw$) στο φάκελο "Certificates E-gate".
</callout>

<callout type="info">
**Η εφαρμογή βρίσκεται στο URL: http://oraxe.mod.gr:8080/ords/f?p=120**
</callout>


==== Install YETHA_CA ====

  - Συνδεόμαστε στο χρήστη ο οποίος θέλει να χρησιμοποιήσει την εφαρμογή.  
  - Ανοίγουμε Internet Explorer και πηγαίνουμε Ρυθμίσεις -> Internet Options -> Content -> Certificates.
  - Στο tab Trusted Root Certifications Authorities κάνουμε import το ΥΕTHA_CA.crt (όταν μας ζητηθεί να κάνουμε browse το certificate που θέλουμε να κάνουμε import, φροντίζουμε να εμφανίζεται ο τύπος αρχείων .crt)
  - Ακολουθούμε τα βήματα με next, μέχρι να εμφανιστεί μήνυμα για επιτυχή εγκατάσταση.

<callout type="warning">
Σε περίπτωση που χρειαστεί, το εγκαθιστούμε και στο tab Intermidiate Certification Authorities.
</callout>


==== Install SEC_PYLH ====

  - Συνδεόμαστε στο χρήστη ο οποίος θέλει να χρησιμοποιήσει την εφαρμογή.
  - Ανοίγουμε Internet Explorer και πηγαίνουμε Ρυθμίσεις -> Internet Options -> Content -> Certificates.
  - Στο tab Personal κάνουμε import το SEC_PYLH.pl2 (όταν μας ζητηθεί να κάνουμε browse το certificate που θέλουμε να κάνουμε import, φροντίζουμε να εμφανίζεται ο τύπος αρχείων .pl2)
  - Ακολουθούμε τα βήματα με next, μέχρι να εμφανιστεί μήνυμα για επιτυχή εγκατάσταση.

<callout type="warning">
Όταν ζητηθεί κωδικός εγκατάστασης, πληκτρολογούμε 1234.
</callout>
