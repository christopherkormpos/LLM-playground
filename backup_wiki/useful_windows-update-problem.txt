====== Πώς Φτιάχνω τις Ενημερώσεις στα Windows 7 Αν Έχουν Κολλήσει ======

Ένα σύνηθες πρόβλημα είναι όταν αυτή η οθόνη παραμένει έτσι κυριολεκτικά για ώρες. Φαίνεται ένας δήθεν έλεγχος για ενημερώσεις, ο οποίος δεν ολοκληρώνεται ποτέ.
Για την αντιμετώπιση αυτού του προβλήματα θα χρειαστεί να εγκαταστήσουμε χειροκίνητα 2 ενημερώσεις. Οι συγκεκριμένες ενημερώσεις υπάρχουν και στον δικτυακό φάκελο, κάτω απ'το win_update.
__Πριν κατεβάσουμε και εγκαταστήσουμε οτιδήποτε, θα πρέπει να βεβαιωθούμε πως δεν τρέχει η υπηρεσία Windows Update.__

Πατάμε το κουμπί των "Windows + R", και στο παράθυρο "Εκτέλεση" γράφουμε: **services.msc**

Στην ενότητα "υπηρεσίες" που θα ανοίξει, βρίσκουμε το Windows Update. Αν στην κατάσταση γράφει "Ξεκίνησε", με επιλεγμένη την υπηρεσία πατάμε το κουμπί του "στοπ" για να τη διακόψουμε.

Εφόσον η διακοπή έγινε σωστά, η στήλη "Κατάσταση" θα είναι πλέον κενή στο Windows Update.

Αν η κατάσταση ήταν κενή εξ' αρχής, αυτό σημαίνει πως η υπηρεσία Windows Update ήταν ήδη σταματημένη. Σε αυτή την περίπτωση, δεν χρειάζεται να κάνουμε τίποτα.

===== Εγκατάσταση ενημέρωσης KB3020369 =====

Το πρώτο βήμα για να φτιάξουμε τις ενημερώσεις στα Windows 7 είναι να εγκαταστήσουμε χειροκίνητα την ενημέρωση KB3020369.

Για τα 32-bit Windows, η αντίστοιχη διεύθυνση είναι:
https://www.microsoft.com/el-gr/download/details.aspx?id=46827 

Εναλλακτικά, για κατέβασμα από το ΤΠΕΝΤ Cloud:
https://192.168.1.150/index.php/s/ia6KoGKYawXYYkC

Και στις δύο περιπτώσεις θα κατεβάσουμε ένα αρχείο τύπου "Μεμονωμένο Πακέτο Microsoft Update".

Τρέχουμε το αρχείο και αποδεχόμαστε την εγκατάσταση. Αν κολλήσει στην αναζήτηση ενημερώσεων και δεν προχωρήσει στο μήνυμα για την εγκατάσταση, σημαίνει πως δεν έχουμε τερματίσει σωστά το Windows Update από τις υπηρεσίες.

Εφόσον όλα πάνε καλά, η εγκατάσταση θα εμφανίσει ένα μήνυμα επιτυχίας.

===== Εγκατάσταση ενημέρωσης KB3172605 =====

Η δεύτερη ενημέρωση που θα χρειαστούμε είναι η KB3172605

32-bit: https://www.microsoft.com/en-us/download/details.aspx?id=53335

Ή από το ΤΠΕΝΤ Cloud:
https://192.168.1.150/index.php/s/c9TBk7BqRqYHY4p

**Αφού την κατεβάσουμε, θα πρέπει να __σταματήσουμε__ την υπηρεσία Windows Update, που την ξεκίνησε η εγκατάσταση της προηγούμενης ενημέρωσης.**

Στη συνέχεια κάνουμε την εγκατάσταση κανονικά.
Στο τέλος της εγκατάστασης θα πρέπει να κάνουμε επανεκκίνηση του υπολογιστή.

Πλέον οι ενημερώσεις στα Windows 7 λειτουργούν
Στην επόμενη εκκίνηση, ξεκινάμε το Windows Update, κανονικά από τον πίνακα ελέγχου, και επιλέγουμε να γίνει έλεγχος για ενημερώσεις.

Πλέον, μέσα σε 5 λεπτά τα Windows 7 βρίσκουν όλες τις απαραίτητες ενημερώσεις.

===== Αποτυχία εγκατάστασης ενημερώσεων =====

Ένα ακόμα σύνηθες πρόβλημα είναι η αποτυχία εγκατάστασης των ενημερώσεων. Σε αυτή την περίπτωση θα πρέπει να γίνουν οι ενημερώσεις ανά μικρά group ώστε να απομονώσουμε το πρόβλημα που προκαλεί την αποτυχία.
