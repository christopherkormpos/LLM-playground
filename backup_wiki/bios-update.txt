====== Bios Update ======

Σε περίπτωση που χρειαστεί να κάνουμε bios update σε κάποιο  HP Compaq 8000 ή HP Compaq 8000 ή HP Compaq 8200 ακολουθούμε τα βήματα:

  - Μπαίνουμε στο BIOS του υπολογιστή για να δούμε την τωρινή version του System BIOS. Για να μπούμε πατάμε F10 πολλές φορές με το που ανοίξουμε το PC. Πιθανότατα θα χρειαστεί να εισάγουμε και τον κωδικό YETHA1.
  - Στην κατηγορία File επιλέγουμε System Information και σημειώνουμε την τιμή του System BIOS.
  - Επισκεπτόμαστε τη σελίδα Λήψης Λογισμικού και Προγραμμάτων Οδηγησης (Software and Driver Downloads) για το εκάστοτε μοντέλο π.χ. https://support.hp.com/gr-el/drivers/selfservice/hp-compaq-8200-elite-small-form-factor-pc/5037931
  - Αφού επιλέξουμε το σωστό λειτουργικό σύστημα (συνήθως Windows 7 32-bit) κάνουμε expand την επιλογή BIOS. Εκεί επιλέγουμε για λήψη το πιο πρόσφατο BIOS ή FIRMWARE update, εφόσον είναι πιο πρόσφατο από αυτό που ήδη υπάρχει στο PC. Προσέχουμε να MHN είναι BIOS Utilities, Microcode κ.λ.π. Συνήθως θα έχει μία παρένθεση.
  - Όταν τελειώσει η λήψη πατάμε δεξί κλικ στο exe αρχείο που κατεβάσαμε και επιλέγουμε **Open with WinRAR**. 
  - Βρίσκουμε το φάκελο *\DOS Flash* και τον ανοίγουμε
  - Μέσα υπάρχει ένα αρχείο με κατάληξη *.ΒΙΝ*. Εισάγουμε ένα usb φλασάκι στον υπολογιστή και κάνουμε drag’n’drop το αρχείο αυτό μέσα στο φλασάκι (**ΠΡΟΣΟΧΗ! __ΟΧΙ__ σε φάκελο**).
  - Βάζουμε το φλασάκι στο PC που θέλουμε να κάνουμε update και μπαίνουμε πάλι στο **BIOS**.
  - Στην κατηγορία ***File*** επιλέγουμε **Flash System ROM** και επιλέγουμε **USB**.
  - Επιλέγουμε το *.BIN* αρχείο και επιλέγουμε **YES/OK/ACCEPT**  στα επόμενα παράθυρα.
  - Τέλος περιμένουμε να ολοκληρωθεί η διαδικασία. 

<callout type="danger">Δεν πρέπει να διακοπεί η διαδικασία του updating  για ΚΑΝΕΝΑ ΛΟΓΟ!</callout>
