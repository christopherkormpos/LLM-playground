====== TITLE: Εγκατάσταση Εκτυπωτή Σε MOD ======

===== Εγκατάσταση USB PRINTER Σε MOD PC =====
==== Περίπτωση 1 - Χρήση Drivers Διαθέσιμων Στα Windows ====

Επιχειρούμε να εγκαταστήσουμε τον εκτυπωτή μέσω των drivers που είναι διαθέσιμοι ήδη στα windows.
  - Devices and Printer 
  - Add a Printer 
  - Add a local printer
  - Use an existing port (USB001 - Virtual Printer Port for USB)
  - Διαλέγουμε εκτυπωτή από την λίστα & ακολουθούμε βήματα installer.

==== Περίπτωση 2 - Χρήση Drivers Από Τον Δικτυακό Φάκελο ====

Θα κάνουμε χρήση drivers που βρίσκονται στον **δικτυακό** φάκελο του ΤΠΕΝΤ.
  - Συνδεόμαστε ως local administrator (hedesk/as.….!@*)
  - Γράφουμε στο windows explorer: ''%%\\10.0.0.15\sw$%%''
  - Πηγαίνουμε *drivers* → *printers* → επιλέγουμε τον κατάλληλο εκτυπωτή & ακολουθούμε τα βήματα του installer

===== Εγκατάσταση Network PRINTER Σε MOD PC =====

Πριν την εγκατάσταση διαδικτυακού εκτυπωτή βρίσκουμε μια διαθέσιμη IP από την σελίδα **Διάφορα > IPs Εκτυπωτών** και επιλέγουμε μια διεύθυνση που είναι διαθέσιμη (**κόκκινο σχέδιο**)

==== Περίπτωση 1 - Χρήση Drivers Διαθέσιμων Στα Windows ====

  - Devices and Printer 
  - Add a Printer 
  - Add a network, wireless or Bluetooth printer
  - The printer that I want isn’t listed 
  - Add a printer using TCP/IP address or hostname 
  - Γράφουμε τη IP που δώσαμε στον εκτυπωτή 
  - Αφού το PC βρεί τον εκτυπωτή, τον επιλέγουμε μαζί με κατάλληλους drivers από την λίστα και ακολουθούμε την διαδικασία

==== Περίπτωση 2 - Χρήση Drivers Από Τον Δικτυακό Φάκελο ====

Θα κάνουμε χρήση drivers που βρίσκονται στον **δικτυακό** φάκελο του ΤΠΕΝΤ.
  - Συνδεόμαστε ως local administrator (hedesk/as….!@*)
  - Γράφουμε στο windows explorer: ''%%\\10.0.0.15\sw$%%''
  - Πηγαίνουμε *drivers* → *printers* → επιλέγουμε τον κατάλληλο εκτυπωτή & ακολουθούμε τα βήματα του installer (επιλέγουμε την εγκατάσταση μέσω δικτύου στις επιλογές του installer και χρησιμοποιούμε την static IP που του έχουμε εκχωρήσει)
